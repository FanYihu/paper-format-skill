# Paper Format Skill

一个专治论文排版焦虑的通用 Agent Skill。

写论文已经够累了，最烦的是最后还要被格式折磨：页边距差 0.2 cm、一级标题字号不对、参考文献缩进怪怪的、图表题注看起来总像“手搓出来的”。这个 skill 的目标很直接：让支持 `.skill` 的 agent 先读懂模板格式，再按规则帮你改 `.docx`，最后再给一份核查报告。

简单说，它是给正在和论文格式要求拉扯的研究生、课程论文选手、毕业论文冲刺人准备的排版辅助工具。

## 用途

- 从 PDF、DOCX 或文字版格式要求中提取论文排版规则
- 生成结构化 `format_rules.json`
- 按确认后的规则修改目标 `.docx`
- 核查页边距、标题层级、正文字体、行距、页眉页脚、图表题注和参考文献格式
- 输出格式核查报告，方便复查修改是否完成

## 解决什么烦点

- 不想一条条翻格式要求：让 agent 先把模板里的可量化规则提出来。
- 不想靠肉眼猜字号和行距：尽量从 PDF 或 DOCX 中提取实际排版参数。
- 不想改完还不知道对不对：应用格式后再跑一次核查，输出报告。
- 不想把论文改成“看起来差不多”：先确认规则，再动目标文档。
- 不想在提交前夜被格式支配：把重复、机械、容易漏的检查交给 skill。

它不是论文内容生成器，也不是替代导师审稿的工具。它主要处理的是论文排版这类“费时间、很机械、但错了很烦”的工作。

## 适合什么时候用

- 学校、学院或期刊给了 PDF / DOCX 模板
- 老师只发了一段文字版格式要求
- 已经写好了论文，但格式需要统一
- 想先提取格式规则，再决定是否套用
- 想在交稿前做一次格式自查

## 安装方法

直接使用根目录下的 `paper-format.skill` 安装包导入支持 `.skill` 的 agent 工具。

如需从源码查看或修改 skill 主体，请编辑 `SKILL.md`。

## 使用方法

在你的 agent 工具中提出类似请求，并同时提供模板和待排版论文：

```text
请按这个 PDF 模板格式，帮我排版我的论文 docx，并输出核查报告。
```

或：

```text
提取这个论文模板的格式规则，先给我确认，再套用到我的论文。
```

Skill 会按三个阶段执行：

1. 分析模板并提取规则
2. 等用户确认后应用格式
3. 重新核查并输出报告
